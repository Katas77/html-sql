<h1>, <h2> и <h3>: Это заголовки разного уровня. Они используются для структурирования контента.
<p>: Этот тег используется для создания параграфов.
class="...": Атрибут class позволяет применить определенные стили к элементам через CSS.
<blockquote>: Используется для выделения цитат.
<ul> и <li>: Эти теги создают ненумерованный список.
<a>: Создает гиперссылку.



CSS:
.normal: Базовые настройки для обычного текста.
.underline: Применяет подчеркивание к тексту.
.italic: Делает текст курсивом.
.bold: Полужирное начертание текста.
.quote: Оформление для цитат.
ul: Настройки для ненумерованных списков.
.link: Стилизует гиперссылки.


Атрибут class в HTML используется для назначения одного или нескольких классов элементам. Классы позволяют применять одинаковые стили к разным элементам страницы через CSS. Вот несколько ключевых моментов об использовании атрибута class:

Назначение класса:Чтобы назначить класс элементу, нужно добавить атрибут class с именем класса в открывающий тег элемента. Например:

<div class="example-class"></div>
Использование нескольких классов:Элемент может иметь несколько классов одновременно. В этом случае имена классов перечисляются через пробел:

<div class="example-class another-class third-class"></div>
Применение стилей через CSS:В CSS классы обозначаются точкой (.), после которой следует имя класса. Например:

.example-class {
    background-color: red;
}
Комбинированные селекторы:Можно комбинировать классы с другими селекторами для более точного применения стилей. Например:

div.example-class {
    font-size: 16px;
}
Каскадность и приоритет:Если у элемента есть несколько классов, и они имеют конфликтующие стили, то применяется правило каскадности и специфичности. Более специфичный селектор имеет больший приоритет.
Пример использования атрибута class и соответствующего CSS-кода:


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <style>
        .header {
            background-color: lightblue;
            padding: 10px;
            text-align: center;
        }

        .content {
            background-color: white;
            padding: 20px;
        }

        .footer {
            background-color: lightgray;
            padding: 10px;
            text-align: center;
        }
    </style>
</head>
<body>
    <div class="header">
        <h1>Заголовок сайта</h1>
    </div>
    <div class="content">
        <p>Основной контент страницы.</p>
    </div>
    <div class="footer">
        <p>© 2023 Все права защищены.</p>
    </div>
</body>
</html>
В этом примере три раздела страницы (header, content и footer) используют разные классы для применения различных стилей.
