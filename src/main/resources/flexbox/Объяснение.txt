Flexbox – это мощный инструмент в CSS, позволяющий легко управлять расположением элементов на странице.
Он особенно полезен для создания адаптивных макетов, когда необходимо гибко изменять размеры и положение элементов в зависимости от размера экрана.
Пояснения:
HTML:
<div class="flex-container">: Контейнер, который служит оболочкой для других элементов и активирует Flexbox.
<div class="item item1">, <div class="item item2">, <div class="item item3">, <div class="item item4">: Четыре блока с разными классами для демонстрации различных возможностей Flexbox.
CSS:
.flex-container: Основные стили для контейнера. Мы включаем Flexbox с помощью display: flex, разрешаем перенос элементов на следующую строку с помощью flex-wrap: wrap и добавляем промежутки между элементами с помощью gap: 10px.
.item: Основные стили для всех элементов. Мы устанавливаем равное распределение свободного пространства с помощью flex-grow: 1 и минимальная ширину элемента с помощью flex-basis: 200px.
.item1, .item2, .item3, .item4: Индивидуальные стили для каждого элемента. Мы демонстрируем различные возможности Flexbox, такие как изменение порядка элементов (order), управление ростом и сжатием элементов (flex-grow, flex-shrink), а также изменение минимальной ширины элемента (flex-basis).
Этот пример показывает базовые возможности Flexbox и их применение для создания гибких и адаптивных макетов. Вы можете экспериментировать с этими свойствами, чтобы лучше понять, как они работают и как их использовать в своих прое
