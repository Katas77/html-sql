Комментарии к CSS:
.my-table { ... } – задаются общие свойства для всей таблицы:
width: 100%; – ширина таблицы равна ширине контейнера.
border-collapse: collapse; – объединяет границы соседних ячеек, делая их едиными.
.my-table th { ... } – стилизация ячеек заголовков:
background-color: #4CAF50; – зеленый фон.
color: white; – белый текст.
padding: 10px; – внутренние отступы.
text-align: left; – выравнивание текста по левому краю.
.my-table td, .my-table th { ... } – стилизация для всех ячеек таблицы:
border: 1px solid #ddd; – тонкая светло-серая граница вокруг каждой ячейки.
padding: 8px; – внутренние отступы.
.my-table tbody tr:nth-child(even) { ... } – используется псевдокласс nth-child(even) для чередования цветов фона строк:
background-color: #f2f2f2; – светло-серый фон для четных строк.
.my-table tbody tr:hover { ... } – изменение фона строки при наведении курсора:
background-color: #ddd; – серый фон при наведении.
