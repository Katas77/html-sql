Пояснения:
HTML:
<div class="container">: Контейнер, который служит оболочкой для других элементов.
<div class="box box1">, <div class="box box2">, <div class="box box3">: Три блока с разными классами для демонстрации различных вариантов настроек размеров и отступов.
CSS:
.container: Основные стили для контейнера. Мы задаём ширину 100%, но ограничиваем её максимальной шириной 800 пикселей. Также центрируем контейнер по горизонтали с помощью margin: 0 auto.
.box: Основные стили для всех боксов. Здесь мы задаём внутренние отступы (padding), внешние отступы снизу (margin-bottom), фоновый цвет и рамку.
.box1, .box2, .box3: Индивидуальные стили для каждого бокса. Мы задаём высоту и ширину, а также используем свойство float для размещения боксов рядом друг с другом.
